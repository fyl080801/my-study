开发环境搭建

开发环境搭建

Flutter 是什么？

  看这里

开发及运行先决条件

  需要先安装 AndroidSDK、Xcode、python2，开发工具 vscode

下载 SDK

  点这里下载，下载完以后解压缩得到 flutter 目录，将该目录下的内容放到/Users/<你的用户名>/Library/Flutter/1.0.0下

配置环境变量(MACOS)

  打开终端，输入

vim ~/.bash_profile

  在打开的编辑器中加入如下内容

export FLUTTER_HOME=/Users/fengyuanliang2/Library/Flutter
export PATH=${PATH}:${FLUTTER_HOME}/1.0.0/bin

  然后刷新一下环境变量

source $HOME/.bash_profile

  之后在终端输入flutter doctor验证是否搭建成功，以及是否具备开发条件
  如果运行完之后出现错误提示，按照提示安装缺少的东西

vscode 插件

  直接在扩展中搜索Flutter

新建项目

  打开 vscode，按command + p打开命令，输入>Flutter:New Project回车，在弹出的输入框中输入一个项目名称，然后选择项目路径完成项目的创建

项目运行

  如果之前开发过 react native 并且能在模拟器中调试，则可以直接运行模拟器，在项目目录下打开终端输入flutter run马上就能看到效果
