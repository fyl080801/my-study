概述

  Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

  GitBook 是一个基于 npm 的命令行工具，可使用将 Markdown 格式的文件转换成 html 格式 ，制作出精美的电子书，GitBook 不是 git 的教程且跟 git 没有任何关系，关于 GitBook 的详细说明可以去看gitbook 的官方文档
